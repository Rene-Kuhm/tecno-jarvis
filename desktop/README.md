# Tecno J.A.R.V.I.S

Asistente de escritorio estilo Iron Man con interfaz HUD futurista, reconocimiento de voz en tiempo real, control del sistema y ejecucion de tareas multi-paso. Tecno--J.A.R.V.I.S.

## Instalacion rapida

Windows:

```bat
install.bat
```

Linux/macOS:

```sh
sh install.sh
```

El instalador crea `.venv`, instala dependencias Python, instala navegadores de Playwright y crea una plantilla `config/api_keys.json` si falta.

## Ejecutar

Windows:

```bat
run.bat
```

Linux/macOS:

```sh
sh run.sh
```
